■ Overview

This app is a digital level (inclinometer) web application that measures the tilt angle of the surface on which a smartphone, tablet, or other device is placed. Since it runs on a web browser, installation is not required.


■ Operating Environment

  • Smartphones / Tablets
    ◆ Android : Available in any version (newer is better)
    Browsers that have been tested: Chrome, Edge, Firefox
    ◆ iPhone / iPad : Versions earlier than iOS 12.2 or versions later than iOS 13
    Browsers that have been tested: Safari
    Unable to access acceleration sensor from browsers in iOS 12.2-13.
    * In addition to the above, it will work on any terminal equipped with a browser that has JavaScript enabled and can access the acceleration sensor.

  • Laptop with an acceleration sensor
    It can be used on a Laptop such as a 2-in-1 equipped with an acceleration sensor and a browser with Javascript enabled.
    As for OS, Windows, MacOS, or Linux are supported.
